------------ 본문
1. 운영체제란 무엇인가?
    컴퓨터(H/W)를 움직이기위한 프로그램 , 프로그램을 실행하기위한
    제일 어른 프로그램이라고 생각하면 될것같습니다.
    우리가 사용하는 많이 사용하는
    Windows를 비롯해서 Dos, Mac OS, OS/2, Unix, Solaris , Linux가 있습니다.

2. 운영체제가 하는 일?
    1. 운영체제는 자원관리기(Resource Manager)로서의 역할을 할 수 있어야 합니다.
       눈이 많이 왔으니까 눈을 치우는 일을 한번 예를 들어 보죠.
       구청이나 시청에서 제설 작업을 위해 인력을 분배하고 투입하게 되죠.
       구청이나 시청이 하는 일이 운영체제가 하는일과 같다고 보면 됩니다.

    2. 사용자와 하드웨어간의 인터페이스라고 보면 됩니다.
       운영체제가 없다면 컴퓨터가 있다고 한들 어떤 처리도 불가능할 것입니다.
       어떤 운영체제로 컴퓨터에 옷을 입히느냐에 따라서.
       컴퓨터의 성능이 결정된다고 봐도 과언이아닐 것입니다.

    3. 감시자, 관리자로서의 운영체제
       이부분이 우리가 지금 공부하고 있는 보안 부분과 연관성이 있습니다.

3. 유닉스 (unix/linux) 란?
1. 1970년 미국의 AT&T Bell 연구소
    2. 창시자 : 켄 톰슨 & 데니스 리치
    3. B언어로 UNIX 개발
    4. 1973년 C언어로 재 탄생
    5. C언어로 쓰여진 최초의 OS
    6. 개방형 운영체제 :
1. 독점 소유 운영체제가 아니다.
        2. 표준 프로그래밍 언어로 작성
        3. 대중적인 많은 idea를 받아들임.


4. 유닉스 시스템의 종류
    1. BSD(Berkeley Software Distribution(Design))
        : 버클리대학에서 개발되고 무료로 배포하기 시작한 유닉스 OS
        : 많은 상용 유닉스 시스템들이 BSD 코드에 기반을 두고 있다.

    2. 시스템 V (SYSV:System Five) 계열 유닉스
        ver.7 (Bell) --> BSD계열 (1980년대) --> 시스템 III, 시스템 V

    3. POSIX(Portable Operating System Interface for Computer Environment)
        개방형 운영체제에 대한 권고 안을 토대로 작성한 개념적인 표준 운영체제

    4. 솔라리스(Solaris)
        썬마이크로시스템즈사의 워크스테이션에 설치되어 대학 및 연구소를 중심으로
        널리 사용되는 상용 유닉스 운영체제의 일종

    5. 리눅스(Linux)
        1. 1991년 리눅스 토발즈가 인터넷 유즈넷 뉴스그룹을 통해 처음 발표한 운영체제
        2. 무료 배포, 유료 & 무료
        3. 386급 PC에서 동작하는 유닉스의 일종
        4. 프로그램 소스 공개 --> 프로그래머가 원하는 대로 특정 기능 추가 기능
        5. 어느 플랫폼에도 포팅이 가능
        6. 99년 5월 국내 5만 이상의 사용자
