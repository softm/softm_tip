------------ 본문
1. 웹 서버의 이해
    1-1. Server란?
        일반적으로 서버라고 하면 다른 프로그램에게 서비스를 제공하는 컴퓨터 
        프로그램을 말한다.

    1-2. 웹서버 (Web server)
        인터네트 상에서 각종 서비스를 제공하기 위한 각종 서버용 프로그램
        웹서버는 웹 페이지가 들어 있는 파일을 사용자들에게 제공하는 프로그램이다.

    1-3 웹서버의 기능 
       클라이언트의 요청을 해석하여 처리, 필요시 CGI에 의해 다른 프로그램과 통신, 
       필요시 데이터베이스에 대한 엑세스 서비스

2. 웹 서버의 종류
     서버의 종류는 수백 가지가 넘게 있는데 뉴스 검색을 위한 뉴스 서버, 채팅을 위한 
     chatting 서버, 전자상거래를 위한 merchant 서버, 카탈로그 서버, 메일 서버, 
     전화번호 서버 등이 있다.
     NCAS, CERN, ETSCAPE, APACHE, WEBSITE, MICROSOFTE,IIS, NIS 등등

3. 웹 서버의 특징.
    3-1. Apache Server
      - NCSA Server(1.3.x)의 개량형으로 성능과 기능면에서 우수 
      - NCSA Server의 기능과 구성을 그대로 유지 
     - 하나의 Port(default:80)에서 일반 Web과 Proxy 동시 지원 
      - 각 기능의 Module화 
      - 가장 많이 사용되는 Web Server 
    # 참고 사이트





    3-2. PWS(Personal Web Server)
       - 개인 사용자 환경에서 동작할 목적으로 MicroSoft사에서 만든 웹 서버
       - 개인 웹 서버용으로 제작되었기 때문에 최대 동시 접속자 수를 10명으로 제한 
       - 웹 서비스는 PWS 프로그램이 실행 중일 때만 실행 됨
